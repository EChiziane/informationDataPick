Como Executar o Projeto

O Projeto
Este projeto é dividido em duas partes principais: o frontend e o backend. Abaixo estão as instruções para configurar e executar cada parte do projeto.

Backend

1. Abra o Terminal ou Prompt de Comando
   Navegue até o diretório `back` onde está o código do backend.

2. Instale o Node.js
   O Node.js é necessário para executar o servidor. Você pode baixá-lo e instalá-lo a partir do [site oficial do Node.js](https://nodejs.org/). Siga as instruções de instalação fornecidas para o seu sistema operacional.

3. Instale as Dependências
   No diretório `back`, instale o Express, que é um framework para Node.js. Execute o seguinte comando no terminal:
   npm install express

4. Execute o Servidor
   Após a instalação, inicie o servidor com o comando:
   node server.js
   O servidor estará rodando e escutando as requisições.

Frontend

1. Abra um Novo Terminal ou Prompt de Comando
   Navegue até o diretório `front` onde está o código do frontend.

2. Instale as Dependências do Frontend
   Instale as dependências necessárias com o seguinte comando:
   npm install

3. Inicie o Servidor de Desenvolvimento
   Execute o servidor de desenvolvimento do Angular com o comando:
   ng serve
   O frontend será iniciado e estará disponível para acesso.

4. Abra a URL no Navegador
   Acesse o projeto frontend no navegador usando a URL:
   http://localhost:4200

Resumo dos Passos

1. Backend:
   - Navegue até o diretório `back`.
   - Instale o Node.js a partir do [site oficial](https://nodejs.org/).
   - Instale o Express com `npm install express`.
   - Execute o servidor com `node server.js`.

2. Frontend:
   - Navegue até o diretório `front`.
   - Instale as dependências com `npm install`.
   - Inicie o servidor com `ng serve`.
   - Acesse o projeto em `http://localhost:4200`.

Certifique-se de seguir cada passo cuidadosamente para garantir que o projeto funcione corretamente. Se você encontrar algum problema, verifique os logs do terminal para mais detalhes sobre erros possíveis.
